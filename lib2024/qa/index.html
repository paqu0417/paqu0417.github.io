  2. Qus: Define version control and give one example.

    Ans: Version control is a system that records changes to a file or set of files over time so that you can recall specific versions later. For the assignments in this class, you will use software source code as the files being version controlled, though in reality you can do this with nearly any type of file on a computer.

    Various version control software :

    • GitHub
    • Mercurial
    • Selenium

  3. Qus: What tag is used to create the title of a web page? Give an example.

    Ans: The tag to create the title of a web page is <title>. <title>Basic HTML Tags</title>.

  4. Qus: What tag is used to create a paragraph in html? Give an example.

    Ans: The tag to create a paragraph in html is <p>. <p>this is a paragraph</p>.

  5. Qus: Identify the odd one from these tags: <span>, <p>, <a>, <i> and state the reason?

    Ans: The odd tag is the span tag because it is the only one that uses CSS.

  6. Qus: What are the tags required to create a navigation list in HTML?

    Ans: <nav>, <li>, <ul>, and <a>.

  7. Qus: What element is used to create images? Name its attributes.

    Ans: The element to create images is <img>. Its attributes are src, alt, width and height.

  8. Qus: Identify the odd one from these tags: <table>, <tr>, <b>, <td> and state the reason?

    Ans: The odd tag out is <b> because it deals with bolding text while the others have to do with creating tables.
